Como usar esta carpeta

Para Claudia (usuario)

  1. Revisar bloqueantes/ primero — nada avanza hasta resolver estas
  2. Luego criticas/ — necesarias antes de constituir la SpA
  3. Luego pendientes/ — necesarias antes de operar

Para un tercero (operador delegado)

  • Entregar SOLO la carpeta operacion/ — es autocontenida
  • Incluye: que hacer por reserva, mensual, anual + links + contactos

Para El Tributarista (agente)

  • Cada comando que detecte una accion humana crea o actualiza un archivo aqui
  • Si genera una recomendacion sobre como abordar acciones, crearla en recomendaciones/ con prefijo R#
  • Si encuentra algo relevante para la operacion recurrente, actualizar operacion/
  • Si algo deja de aplicar, eliminarlo de operacion/
